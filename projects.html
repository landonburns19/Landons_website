Interviewing Ghosts

Interviewing Ghosts is a local AI powered unity game. It uses Ollama and langchain to support a user's dialogue between seperate three characters: Alonso, Napoleon, and Victoria. The goal is to tease out details about an art thief from each character. If the user finds the right details, they can win by guessing the correct suspect. The details are stored in a csv file that the models can access.

3Dio

3Dio is a multithreaded 3d object engine. It can create shapes out of any set of polygons by using triangles. A developer can then use the base system to easily program interactions between objects.

Haskell project

Haskell project is a coding language that can execute programs for a specified amount of time. It works by compiling a textfile with all the programs, file names, and times. It then uses forkIO with a monadic function to create a thread for each specified file.
